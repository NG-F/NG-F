# NG+F (New Game plus Fork): 構想

## 【目的】
本プロジェクトは、**長期目標を維持できる組織の設計原理**を研究し、その成果を社会へ還元することを目的とする。

多くの組織は、創設時の理念から次第に逸脱し、組織の維持そのものが目的化（ミッション・ドリフト）する傾向にある。本プロジェクトはその原因を工学的に研究し、より持続可能で透明性が高く、目的を維持し続けられる組織の実現を目指す。

## 【基本理念】
* **組織も研究対象である**：組織構造そのものを分析・改善の対象とする。
* **定款は組織の設計図である**：ルールをコードのように扱う。
* **ガバナンスは工学的に研究可能である**：意思決定の仕組みを科学する。
* **失敗した組織からも学ぶ**：解散や衰退のプロセスをデータ化する。
* **組織の進化を記録する**：変化の過程を透明に保つ。
* **財団自身も研究対象とする**：本組織の運営データも公開する。
* **組織は「フォーク（分岐）」し「分散」可能である**：理念を複数の法人や形態に継承させる。
* **コード（定款）の公開**：組織の死を超えて、理念を次世代へ存続させる。

---

## 【主要研究分野】
本プロジェクトは**「組織設計学」**を基盤領域とし、以下の下位分野の研究を行う。

### 1. 組織設計学
組織の目的・構造・権限配置・意思決定過程・長期存続の条件を研究。
* **対象例**: 財団、NPO、一般社団法人、協同組合、学会、市民コミュニティ

### 2. ガバナンス工学
理念を維持するための制度設計の研究。
* **研究内容**: 権力集中防止、監査制度、乗っ取り耐性、意思決定構造、世代交代、情報公開、定款収集・比較分析、利益相反規定分析

### 3. 制度進化学
組織の世代交代・分岐・継承・制度変化の研究。
* **研究内容**: 定款変更履歴分析、長寿組織研究、解散組織研究、ミッションドリフト研究、組織進化の追跡
* **例**: `財団A` → `財団B` → `財団C` といった組織の変化を記録し、有効な制度を検証する。

### 4. 分散組織研究
単一法人への依存を減らし、理念の継承可能性を高める研究。
* **研究内容**: 組織のオープンソース化、多法域展開、資産分散管理、自己解体と再構成、理念ネットワーク設計

---

## 【成果物】
1. **定款アーカイブ**: 定款、規約、憲章、ガバナンス資料の収集と保存。
2. **組織データベース**: 理事会・評議員制度、解散条件、改正条件等の比較可能な整理。
3. **定款AI**: 定款や規則を読み込み、事業計画との整合性を確認する（適合・要審査・矛盾の判定）。
4. **財団設計支援**: 新たな組織の設立（定款作成、ガバナンス設計、リスク分析）の支援。
5. **財団OS (Foundation Open Source)**: 信頼性の高い組織をデプロイ（構築）できるパッケージ。
   * 憲章・定款テンプレート（Markdown/GitHub形式）
   * 各国の法制度に対応した適合モジュール
   * 運用ログの公開プロトコル
6. **分散型アーカイブ**: 財団が消滅しても、設計図とデータが永久に残る仕組み（IPFS等の活用）。

---

## 【特徴】
* **組織そのものが成果物**: 通常の研究機関が論文を出すのに対し、本プロジェクトは「より良い組織設計（OS）」を成果とする。
* **自己修復性と継承性**: 単一の法人格に依存しない「理念のネットワーク」として活動。特定の法人が解散しても、別の法域の組織が機能を継承する仕組みを追求する。

## 【長期目標】
人類が長期目標を維持できる組織の設計原理を発見し、その知見を公開する。

1. **記録する** (Record)
2. **分析する** (Analyze)
3. **設計する** (Design)
4. **実証する** (Verify)
5. **継承する** (Inherit)

この循環を継続し、次世代の組織基盤を構築する。

---

### 補足：本構想における「フォーク」について
本構想における「フォーク」は、必ずしも新たな財団法人の設立を意味しません。コストや法制度を考慮し、プロジェクト、研究会、NPO、DAOなど最適な形態を選択します。
しかし、財団という形態を重視するのは、それが「人の集合体」ではなく**「目的のために拠出された財産」**を基盤とするためです。これにより、短期的な人気や利益に左右されず、「少数であっても守る価値のある目的」を長期的に継承し続けることが可能になります。
